fox-java
========

Java bindings for FOX - Federated Knowledge Extraction Framework


[Here] (https://github.com/renespeck/fox-java/blob/master/src/main/java/org/aksw/fox/binding/java/Examples.java) you can find an example.

    <repositories>

    </repositories>

        </dependency>
    </dependencies>
